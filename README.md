The Weather Data REST API is a Spring Boot application that loads weather data from a CSV file, stores it in a MySQL database, and exposes RESTful endpoints to retrieve weather records and monthly temperature statistics. The application follows a clean layered architecture consisting of Controller, Service, and Repository layers, ensuring proper separation of concerns and maintainable code structure.
Built using Java 21, Spring Boot 3, Spring Data JPA, Hibernate, and MySQL, the system automatically ingests CSV data at startup using a CommandLineRunner and prevents duplicate entries by inserting records only if the table is empty. The API allows users to fetch weather data by date, by month, and to obtain aggregated monthly statistics (maximum, minimum, and average temperature) for a given year.
The project demonstrates RESTful API design, DTO-based response modeling, database integration, and business logic implementation, making it suitable for backend development practice, technical assessments, and portfolio presentation.
